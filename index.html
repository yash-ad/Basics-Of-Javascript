Double and single quotes are SIMPLE quotes there is practically no difference between them in javascript

Backticks are exteneded functionality quotes they allow us to embed variables and expresssions intpo a string by wrapping.

In javascript the textual data is stored as a string and there is no seperate type for a single character

We should be aware that strings in javascript are encoded using UTF-16.that is each character has a coressponduing numeric code
